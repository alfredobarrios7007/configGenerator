GIT

  • Merge
    • 
git checkout master
git pull origin master
git merge <branch>
git push origin master


  • Create Branch
    • 
git checkout -b <branch>



  • Delete branch locally
    • 
git branch -d localBranchName


  • Delete branch remotely
    • 
git push origin --delete remoteBranchName
